What makes this different from Chrome?

Chrome is made by Google. Google is the world's largest advertising company. Chrome's privacy protections are deliberately limited because blocking too much tracking hurts Google's core business. Every "privacy feature" in Chrome has exceptions that still allow Google to track you.

Beldex Browser is built on Chromium (the open-source code that Chrome is based on) so all your existing websites, extensions, and bookmarks work exactly the same. The difference is that all Google services and telemetry have been removed and replaced with genuine privacy protections.

Built on Chromium works with everything

Since Beldex Browser uses the same Chromium engine as Chrome, every website that works in Chrome works identically in Beldex Browser. You do not lose any functionality. You only gain privacy.

What about Brave Browser?

Brave is also a privacy browser built on Chromium. Beldex Browser goes further by integrating natively with the Beldex ecosystem: a built-in BDX wallet, .bdx domain resolution through BNS, and optional BelNet routing all features that Brave does not have.

Section 04

Regular browsers cannot
open .bdx websites.
Beldex Browser can.

A .bdx domain is a website address registered on the Beldex blockchain. It cannot be censored, taken down, or seized by any authority. Regular browsers like Chrome cannot open these addresses they have no way to resolve a blockchain domain. Beldex Browser has the Beldex Name System (BNS) built in, so .bdx addresses work exactly like normal website URLs.

Why .bdx websites are different

Regular domain (google.com)

Registered through ICANN-authorized registrars. Can be suspended, seized, or redirected by governments and courts. When the registrar complies with a takedown order, the domain disappears.

.bdx domain (privacynews.bdx)

Registered on the Beldex blockchain for 5 BDX. The record is cryptographically permanent no registrar, no takedown authority, no ICANN. The domain exists as long as the Beldex network exists.

Try typing any .bdx address in the browser bar below. Beldex Browser uses BNS (Beldex Name System) to look up the address directly on the blockchain and open the website.

How does BNS resolution work inside the browser?

When Suhail types privacynews.bdx in Beldex Browser, the browser sends a query directly to the Beldex blockchain (not to any DNS server). The blockchain returns the BelNet service address registered under that name. The browser then connects to that address through BelNet. The entire process happens automatically from Suhail's perspective, it works exactly like typing a normal URL.

Beldex Browser includes optional BelNet integration. When you enable BelNet routing, all your browser traffic is automatically routed through the BelNet decentralized VPN through 3 relay nodes before reaching the internet. Three privacy levels you choose

Level 1 Standard Privacy

Ads and trackers blocked. HTTPS enforced. Cookie control active. Your IP address is still visible to websites. Good for everyday browsing where speed matters.

Level 2 Private Window Mode

Everything in Level 1 plus no browsing history saved, no cookies retained after the session, and enhanced fingerprint protection. Your IP is still visible. Good for private sessions on shared devices.

Level 3 BelNet Routing Active

Everything in Level 2 plus your traffic routes through the BelNet decentralized VPN. Your real IP address is completely hidden from every website you visit. Maximum privacy some speed reduction.
