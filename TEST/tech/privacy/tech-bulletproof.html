What is Bulletproof++?

Bulletproof++ is an advanced zero-knowledge proof system that verifies transaction amounts are valid (positive and within range) without revealing the actual values.

In privacy blockchains like Beldex, range proofs are essential. When transaction amounts are hidden by RingCT, the network needs to guarantee no one creates coins out of thin air.

Key Achievement: Deployed via hardfork on December 7, 2025 ‚Äî one of the most significant protocol upgrades in Beldex history.

How It Works

Bulletproof++ builds on the original Bulletproofs with key improvements:

Range Proof Generation

Generates compact proof that each hidden amount is between 0 and 2^64 ‚Äî proving no inflation without revealing the value.

Logarithmic Proof Size

Proof size grows logarithmically with outputs, not linearly.

Batch Verification

Multiple proofs verified together as a batch, significantly speeding up block validation.

No Trusted Setup

Why Bulletproof++ Matters

  • Transaction scalability: Smaller proofs mean smaller transactions, reducing bandwidth and storage.
  • Lower fees: Smaller sizes translate to lower fees for everyday private transactions.
  • Faster sync: New nodes can verify blockchain history faster, improving decentralization.
  • Foundation for EVM: Efficiency gains offset overhead from EVM smart contract integration.
