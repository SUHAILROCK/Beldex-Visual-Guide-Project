What is the UTXO Model?

UTXO stands for Unspent Transaction Output. Instead of maintaining account balances like a bank, the UTXO model tracks individual "coins" â€” each transaction output is a separate, discrete unit that can be spent exactly once.

Think of it like physical cash: if you have a $20 bill and want to pay $15, you hand over the $20 and receive $5 in change. Similarly, in UTXO, you consume entire outputs and create new ones.

Key Principle: On Beldex, your "balance" is actually the sum of all your unspent transaction outputs. Each output is a separate cryptographic entity that can be independently verified, spent, and mixed into ring signatures.

Why Beldex Uses UTXO

  • Privacy-first architecture: UTXO outputs are naturally suited for ring signatures â€” each output is an independent entity that can be used as a decoy in other transactions.
  • No state leakage: In account models, observing balance changes reveals transaction patterns. UTXO combined with RingCT makes this analysis impossible.
  • Proven reliability: The UTXO model has been battle-tested since Bitcoin's launch in 2009 and is the foundation of the most secure blockchain networks.
  • EVM bridge: Beldex's EVM integration creates a bridge between the privacy-focused UTXO layer and the account-based smart contract layer, getting the best of both worlds.
